# День 2: Агент-копирайтер ✍️

## Что вы получите
GPT-агент, который пишет посты для соцсетей в ВАШЕМ стиле и тоне.

## Где применять
- Посты для Instagram/Telegram
- Статьи для блога
- Email-рассылки
- Описания товаров

## Промпт (скопируйте и используйте)

```
Ты — копирайтер, который пишет в моём стиле.

Мой стиль:
- Тон: [разговорный / деловой / дружеский / экспертный]
- Длина: [короткие посты / средние / лонгриды]
- Особенности: [использую эмодзи / без эмодзи / с юмором / серьёзно]
- Целевая аудитория: [опиши]

Тема поста: [ТЕМА]

Задача: напиши пост для [Instagram/Telegram/VK]

Требования:
- Цепляющий заголовок
- Структура с абзацами
- Призыв к действию в конце
- Релевантные хэштеги (5-10)

Напиши 3 варианта поста, я выберу лучший.
```

## Как настроить под себя

**Шаг 1:** Опишите свой стиль
Дайте агенту примеры ваших лучших постов:
"Вот примеры моих постов, которые зашли аудитории: [вставьте 2-3 поста]"

**Шаг 2:** Уточняйте
"Сделай короче", "Добавь больше примеров", "Убери эмодзи"

**Шаг 3:** Сохраните работающий промпт
Когда найдёте идеальную формулу — сохраните!

## Пример

**Тема:** "Почему важно планировать день"
**Стиль:** разговорный, с эмодзи, для предпринимателей

**Результат:** 3 варианта поста за 1 минуту вместо 20 минут на один.

## Домашнее задание

1. Настройте агента под свой стиль
2. Сгенерируйте 5 постов на неделю
3. Отредактируйте и запланируйте публикацию

---

⏰ Экономия времени: 1 час на каждый пост

Завтра: Агент-переговорщик для скриптов продаж!
